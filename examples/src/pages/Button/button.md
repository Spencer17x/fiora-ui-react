# API

# Button

|  属性   | 说明  |  类型   | 默认值  |
|  :----  | :----  |  :----  | :----  |
| type  | 设置按钮类型 | 'default' \| 'primary' \| 'dashed' \| 'danger' \| 'link' | 'default' |
| size  | 设置按钮大小 | 'small' \| 'middle' \| 'large'  | - |
| disabled  | 按钮失效状态 | boolean  | false |
| icon  | 设置按钮的图标组件 | string  | - |
| loading  | 设置按钮载入状态 | boolean  | false |
| shape  | 设置按钮形状 | 'circle'  | - |
